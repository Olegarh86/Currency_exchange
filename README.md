# Проект обмен валюты.
---
## Суть проекта:
[Exchanger](http://exchanger-app.ru/) это калькулятор обмена валют. В него можно добавлять валюты и обменные курсы между существующимим валютами.
При старте присутсвуют только 11 валют:
- USD US Dollar
- EUR Euro
- AMD Armenian Dram
- ARS Argentine Peso
- AUD Australian Dollar
- BRL Brazilian Real
- MXN Mexican Peso
- ZWL Zimbabwe Dollar
- SEK Swedish Krona
- RUB Russian Ruble
- JPY Yen

Согласно стандарту [ISO_4217](https://ru.wikipedia.org/wiki/ISO_4217) коды валют состоят из трёх символов: латинские буквы или цифры.
Максимальная допустимая длинна названия валюты 50 символов.

## Обмен валют
Обмен валюты A в валюту B может происходить по 3-м сценариям:
- в базе данных существует обменный курс A/B
- в базе данных существует обратный обменный курс B/A
- в базе данных существует обменный курс A/USD или USD/A и B/USD или USD/B

## Ограничения
- можно добавть только существующий код валюты [ISO_4217](https://ru.wikipedia.org/wiki/ISO_4217) состоящий из трёх символов
- длинна названия валюты должна быть не более 50 символов
- длинна символа валюты не должна превышать 3 символа
- невозможно добавить валюту без имени
- обменный курс может содержать не больше 6 знаков после запятой

## Тестирование API в IntelliJ IDEA

В проекте есть файл `rest-api.http` для тестирования всех эндпоинтов.

1. Откройте проект в IntelliJ IDEA.
2. Откройте файл `rest-api.http`.
3. Запускайте запросы по кнопке **Run** слева от каждого запроса.
